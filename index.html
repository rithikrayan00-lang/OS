Monolithic Architecture
Description
A single, large kernel that handles all OS functions.
Key Characteristics
• Single executable kernel
• All services in kernel space
• Direct function calls
• Fast communication
Pros & Cons
✓ Advantages
  • Fast performance
  • Simple design
✗ Disadvantages
  • Hard to maintain
  • Single failure point

Linux, DOS, Windows (partially)
